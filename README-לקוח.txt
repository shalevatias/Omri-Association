============================
מערכת ניהול עמותת עמרי - הוראות התקנה והפעלה ללקוח
============================

1. דרישות מקדימות:
-------------------
- מחשב Windows
- חיבור לאינטרנט
- חשבון Google עם הרשאות לעריכת הגיליון

2. התקנת Python (אם אין):
--------------------------
- פתח את האתר: https://www.python.org/downloads/
- הורד והתקן Python (סמן V על "Add Python to PATH" בזמן ההתקנה)

3. התקנת המערכת:
-----------------
- חלץ את כל הקבצים מהקובץ Omri-Association.zip לתיקייה במחשב.
- לחץ פעמיים על הקובץ install.bat
- המתן לסיום ההתקנה (המערכת תתקין את כל מה שצריך אוטומטית)

4. הפעלת הדשבורד:
------------------
- בסיום ההתקנה ייפתח הדשבורד אוטומטית בדפדפן (http://localhost:8501)

5. חיבור ל-Google Sheets (פעם ראשונה בלבד):
---------------------------------------------
- בפעם הראשונה תתבקש להדביק מפתח Google (service account key)
- לחץ על הכפתור במערכת, פתח את עמוד יצירת מפתח ב-Google Cloud
- צור מפתח חדש (JSON), העתק את כל התוכן והדבק במערכת
- לחץ "בדוק ושמור מפתח חדש"

6. תמיכה:
----------
בכל שאלה או תקלה ניתן לפנות לתמיכה של עמותת עמרי.

בהצלחה!
